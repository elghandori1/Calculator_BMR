# BMR Calculator

## Overview
This project is a **Basal Metabolic Rate (BMR) Calculator**. It allows users to calculate their BMR based on their personal details such as age, gender, weight, and height. The BMR is an estimate of the number of calories your body needs to perform basic life-sustaining functions.

## Features
- User-friendly interface.
- Input fields for age, gender, weight, and height.
- Calculates BMR based on the provided inputs.
- Displays the result in an easy-to-read format.

## Files in the Project
- **pageBMR.html**: The main HTML file containing the structure of the calculator.
- **pageBMR.css**: The CSS file for styling the calculator.
- **pageBMR.js**: The JavaScript file for handling the logic and calculations.
- **images/**: A folder containing any images used in the project.

## How to Use
1. Open the `pageBMR.html` file in any modern web browser.
2. Enter your details (age, gender, weight, and height) into the input fields.
3. Click the "Calculate" button to see your BMR result.

## Technologies Used
- **HTML**: For structuring the web page.
- **CSS**: For styling the web page.
- **JavaScript**: For implementing the BMR calculation logic.
